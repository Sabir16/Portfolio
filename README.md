# Portfolio - KADDOURI Sabir

## Fichier CV requis pour la page « À propos »

Pour que les boutons **Consulter CV** et **Télécharger CV** fonctionnent, placez votre CV au format PDF à la racine du projet sous le nom :

- **`CV.pdf`**

Emplacement : `d:\Portfolio\CV.pdf`

Si votre fichier a un autre nom, modifiez dans `Apropos.html` les attributs `href` et `download` des deux liens pointant vers le PDF.
